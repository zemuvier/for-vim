for-vim
=======

Repo with my vim plugins

How to install:
---------------

 *. Delete ~/.vim/bundle/ folder (if you have it)
 *. Clone this repo and navigate to it
 *. Run ./install.sh


Available languages:

 *. Python
 *. Go
 *. Bash
 *. Ruby
 *. JSON

You can enable HTML, CSS, JS, Rust, Clojure and Erlang languages using file ``vimrc``.
