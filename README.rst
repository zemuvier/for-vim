for-vim
=======

Repo with my vim plugins

How to install:
---------------

 1. Delete ~/.vim/bundle/ folder (if you have it)
 2. Clone this repo and navigate to it
 3. Run ./install.sh


Available languages:
--------------------

 1. Python
 2. Go
 3. Bash
 4. Ruby
 5. JSON

You can enable HTML, CSS, JS, Rust, Clojure and Erlang languages using file ``vimrc``.
